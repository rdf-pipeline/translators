Patient Appointment (44_2) Schema Properties:


This is the main file that is used by PAS to store all Patient appointment
info for each appt booked by scheduling clerks. When End of Day Processing
is done this is also the file that is modified to indicate whether a patient
had a status of Kept, No-Show, Pending, or Cancelled.
