Order (101) Schema Properties:


This file contains all orders, both inpatient and outpatient for all
patients in the system. The fields are grouped as follows:

Subscript 0 - all fields whose primary purpose is to uniquely identify
an order and link it to the patient, encounter and episode.

Subscript 1 - the fields responsible for tracking order signatures and
authorizations.

Subscript 2 - all order status flags and status tracking fields. Includes
order priority flags (Eg.STAT).

Subscript 3 - order modification fields

Subscript 5 - multiple sequential orders

Subscript 50 - subfile for tracking secondary orders

Subscript 51 - subfile to track Warnings issued

Subscript 52 - multiple subfile for requisitioning and resulting orders

Subscript 53 - subfile for tracking changes in order status

Subscript 101 - multiple subfile for tracking QA 'failed events'

Subscripts 110-114 - Laboratory orders

Subscripts 115-119 - Nursing/Intraward orders

Subscripts 120-129 - Pharmacy

Subscripts 125-129 - Diet orders

Subscripts 130-134 - Radiology orders

Subscripts 150-154 - Ancillary orders

Subscripts 160-164 - ADT orders
