Provider (6) Schema Properties:


The PROVIDER file contains personal and professional data about the
people who provide patient care services at this site. These include
not only physicians, but also, for example, dentists, nurses, therapists,
pharmacists, and technicians. This file also contains QA data
concerning the provider's professional credentials.

The Master File Primary Key (Unique ID) for this file:
IEN\NAME(.01)\99STF
