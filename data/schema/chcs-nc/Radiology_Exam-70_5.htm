Radiology Exam (70_5) Schema Properties:


This file contains Radiology exam information for patient procedures.
This file is the focal point of the radiology module. Imaging data for
Radiology, Nuclear Medicine, and Radiation Therapy procedures is stored here.

OrderLabelTypeDescription
1Exam No. (exam_no-70_5) STRINGThis field contains the number of the radiology exam. The CHCS creates the
exam number automatically at the time the radiology exam is ordered. A
radiology exam number is an eight-digit number composed of two parts: the
last two digits of the current year followed by a six-digit, sequential
number. For example, the exam number 93000345 indicates that this was the
345th radiology exam ordered in 1993.
2Name (name-70_5) POINTER (Patient-2)This field contains the name of the patient for whom this radiology exam
was ordered. The CHCS automatically enters the name of the patient you
select during the order entry process.

The CHCS automatically enters the name of the procedure you selected during
the order entry process in this field.
4Exam Status (exam_status-70_5) POINTER (Examination_Status-72)This field contains the status (for example, ARRIVED or EXAMINED) of this
radiology exam.
5Order Task Number (order_task_number-70_5) POINTER (Order_Task-107)This field contains the order task number. The CHCS automatically enters
the order task number in this field.
